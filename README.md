# Catbrary

An app for keeping track of the cats you see around, and for getting info about multiple cat breeds

![Kotlin](https://img.shields.io/badge/-Kotlin-7F52FF?logo=kotlin&logoColor=white)




## Features

- Material 3 Design
- Cat breeds info
- Info about cat breeds
